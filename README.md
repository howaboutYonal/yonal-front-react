## 📍프로젝트 목표 및 상세 설명

- 프로젝트 목표 : 멤버들의 비는 시간을 파악해 약속 시간을 정해주는 웹앱
- 상세 설명 :
    
    모임 주최자가 미팅 목적, 날짜를 설정 후 링크를 통해 멤버들에게 보내면 멤버들이 가능한 날짜를 선택 후 제출
    
    투표가 완료되면 투표 결과를 공유할 수 있으며 그렇게 정해진 일정들을 Yonal 메인 홈페이지에서 관리 가능
    
    구글 로그인으로 이용할 수 있으며 멤버들은 로그인 불필요
